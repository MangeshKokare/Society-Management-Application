# 🏢 Society Management Application

A full-featured web-based Society Management System built using **Django**.  
This application helps residential communities manage daily operations digitally with role-based access and secure workflows.

---

## 🚀 Key Features

### 👤 Role-Based Access System
- Society Admin
- Residents
- Security Guards
- Service Providers

Each user role has a dedicated dashboard and specific permissions.

---

## 🏠 Resident Features

- Visitor Management
- Parcel Notifications
- Community Posts
- Vehicle Registration
- Emergency Contacts
- Service Booking
- Notices & Announcements
- Profile Management
- Real-time Notifications

---

## 🛡 Security Guard Features

- Visitor Entry Management
- Parcel Handling
- Patrol Tracking
- Shift Reports
- Leave Requests
- Attendance Tracking
- Chat with Admin
- Guard Notifications

---

## 🏢 Society Admin Features

- Manage Residents
- Manage Guards
- Approve / Reject Leave Requests
- Publish Notices
- Analytics Dashboard
- Society Settings
- Notifications Management
- Guard Attendance Overview

---

## 🔧 Service Provider Features

- Service Provider Registration
- Client Management
- Service Requests Handling
- Notifications
- Profile Management

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (Development)
- **Frontend:** HTML, CSS, Bootstrap
- **Authentication:** Django Authentication System
- **Role Management:** Custom Role-Based Access Control
